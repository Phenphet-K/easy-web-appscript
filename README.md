# Easy Web AppScript - ระบบเว็บไซต์โรงเรียนออนไลน์อัจฉริยะ (Google Apps Script CMS)

ระบบเว็บไซต์โรงเรียนและพอร์ทัลผู้ดูแลระบบ (Admin Portal) แบบ Single Page Application (SPA) ที่พัฒนาขึ้นโดยใช้ **Google Apps Script** เป็นระบบหลังบ้าน ร่วมกับฐานข้อมูล **Google Sheets** และพื้นที่จัดเก็บรูปภาพบน **Google Drive** พร้อมผสานพลังเทคโนโลยีปัญญาประดิษฐ์ (AI) ในการช่วยเขียนเนื้อหาและเจนภาพประกอบ

---

## 🌟 คุณสมบัติเด่น (Features)

*   **🎨 Dynamic Theme Styling CMS:** ปรับเปลี่ยนชุดสีหลัก (Primary), สีรอง (Secondary), สีเน้น (Gold), สีพื้นหลัง, รูปแบบตัวอักษร (Google Fonts) และขนาดอักษรของทั้งเว็บไซต์ได้แบบเรียลไทม์ผ่านหน้าการตั้งค่าหลังบ้าน
*   **📱 Fully Responsive Design:** การแสดงผลบนมือถือที่ถูกปรับปรุงอย่างสมบูรณ์แบบ แผงควบคุมเลื่อนแถบด้านข้างได้ ตารางขยายตัวแบบเลื่อนตามแนวนอน ฟอร์มจัดเรียงตัวสวยงามอ่านง่ายบนหน้าจอมือถือ
*   **🤖 Smart AI Text Writer (Gemini & Groq):** ผู้ช่วยอัจฉริยะในการเขียนบทความและประกาศประชาสัมพันธ์:
    *   **Google Gemini API:** เรียกใช้งานผ่านโมเดล `gemini-2.5-flash` สำหรับความเข้าใจบริบทที่ดีเยี่ยม
    *   **Groq API:** สลับไปใช้ความเร็วสูงของ `llama-3.3-70b-versatile` บนเครือข่ายของ Groq ได้ทันที
*   **🖼️ Smart AI Image Generator (Pollinations.ai ฟรี 100%):** 
    *   ระบบสร้างรูปภาพประกอบบทความผ่าน **Pollinations.ai** โดยตรง (ไม่มีข้อจำกัดเรื่องโควตา และไม่ต้องใช้ API Key)
    *   ดึงรูปภาพในอัตราส่วนแนวนอน (Landscape 16:9) เข้ามาบันทึกลงใน Google Drive โฟลเดอร์ของโรงเรียนโดยอัตโนมัติ เพื่อเป็นแหล่งเก็บรูปภาพที่ถาวรและรวดเร็ว
*   **📁 Google Drive Media Storage:** แดชบอร์ดจัดการภาพแบบหลายรูป (Multi-Image Uploader) และปุ่มอัปโหลดรูปภาพ `🖼️` แทรกรูปภาพตรงตำแหน่งเคอร์เซอร์ลงใน Rich Text Editor โดยภาพทั้งหมดจะถูกบันทึกเข้าโฟลเดอร์ Google Drive ของโรงเรียนโดยอัตโนมัติ
*   **📊 Full CRUD Admin Dashboard:** แผงควบคุมสำหรับจัดการเนื้อหาทั้งหมดในจุดเดียว:
    *   **ข่าวประชาสัมพันธ์ (News):** ข้อมูลข่าวประชาสัมพันธ์ หน้าปก หมวดหมู่
    *   **ภาพกิจกรรม (Activities):** ภาพถ่ายแกลเลอรี่กิจกรรมโรงเรียน
    *   **ปฏิทินกิจกรรม (Events):** ปฏิทินกิจกรรมและการนัดหมาย
    *   **ประกาศอัปเดต (Updates):** บันทึกประกาศและการอัปเดตสถานะของโรงเรียน
    *   **คณะผู้บริหาร (Board):** จัดการรายชื่อ บทบาท และรูปภาพคณะผู้บริหาร
    *   **กล่องข้อความ (Messages inbox):** รับข้อความติดต่อจากฟอร์มหน้าบ้าน พร้อมระบบบันทึกโน้ต
*   **🔒 Security Session:** ระบบยืนยันตัวตนความปลอดภัยสูงแบบ Session Token อายุใช้งาน 8 ชั่วโมง

## 📂 โครงสร้างไฟล์ในโครงการ

*   `code.gs` - โค้ดฝั่งหลังบ้าน (Server-side JavaScript) สำหรับจัดการ Routing, CRUD ฐานข้อมูล และการเชื่อมต่อ AI APIs
*   `index.html` - หน้าโครงร่างหลักและส่วนควบคุมโครงสร้าง SPA
*   `home.html` - หน้าแรกของโรงเรียน (ข่าวเด่น, แกลเลอรี่ภาพกิจกรรม, ทำเนียบผู้บริหาร, แผนที่โรงเรียน และฟอร์มติดต่อ)
*   `about.html` - หน้าข้อมูลประวัติ วิสัยทัศน์ และคณะผู้บริหารโรงเรียน
*   `updates.html` - หน้ารวบรวมประกาศข่าวสารและบันทึกอัปเดต
*   `contact.html` - หน้าข้อมูลช่องทางการติดต่อและแผนที่ฝังตัว
*   `admin.html` - ส่วนประกอบ UI แดชบอร์ดและฟอร์มสำหรับผู้ดูแลระบบ
*   `css.html` - สไตล์ชีตรวมของเว็บไซต์ทั้งหมด (พร้อมสไตล์แบบ Responsive)
*   `javascript.html` - สคริปต์หน้าบ้าน ควบคุมการแสดงผล SPA, Rich Text Editor และการทำ Fallback AI

## 🚀 การติดตั้งและจัดวางระบบ (Installation & Deployment)

### ขั้นตอนที่ 1: เตรียมฐานข้อมูลและโฟลเดอร์รูปภาพ
1.  สร้าง **Google Sheets** ใหม่ 1 ไฟล์
2.  สร้าง **Google Drive Folder** ใหม่ 1 โฟลเดอร์สำหรับเก็บภาพ และแชร์แบบ **"ทุกคนที่มีลิงก์มีสิทธิ์อ่าน"**
3.  คัดลอก **ID ของ Google Sheets** และ **ID ของโฟลเดอร์ Google Drive**
4.  เปิดไฟล์ `code.gs` และนำค่าดังกล่าวไปใส่ในส่วนหัวของไฟล์:
    ```javascript
    const CONFIG = {
      SHEET_ID: 'รหัส_GOOGLE_SHEETS_ของคุณ',
      DRIVE_FOLDER_ID: 'รหัส_โฟลเดอร์_GOOGLE_DRIVE_ของคุณ',
      SCHOOL_NAME: 'ชื่อโรงเรียนของคุณ',
      SESSION_HOURS: 8
    };
    ```

### ขั้นตอนที่ 2: ตั้งค่าบน Google Apps Script
1.  ใน Google Sheets ของคุณ ไปที่เมนู **ส่วนขยาย (Extensions) > Apps Script**
2.  ลบโค้ดเริ่มต้นทั้งหมดออก
3.  สร้างไฟล์โค้ดใหม่ที่มีชื่อดังต่อไปนี้ให้ตรงกับโครงการ:
    *   `Code.gs` (วางโค้ดจาก `code.gs`)
    *   `index.html` (วางโค้ดจาก `index.html`)
    *   `home.html`
    *   `about.html`
    *   `updates.html`
    *   `contact.html`
    *   `admin.html`
    *   `css.html`
    *   `javascript.html`
4.  กด **บันทึกโครงการ (Save)**

### ขั้นตอนที่ 3: เริ่มต้นระบบฐานข้อมูล (Database Initialization)
1.  ในหน้าต่าง Apps Script เลือกฟังก์ชันชื่อ **`setupSpreadsheet`** จากรายการด้านบน
2.  กดปุ่ม **เรียกใช้ (Run)**
3.  ระบบจะสร้างชีตย่อยและโครงสร้างคอลัมน์ข้อมูลที่จำเป็น (`News`, `Activities`, `Events`, `Settings`, `Admins`, `Board`, `Messages`, `Updates`) พร้อมแอดมินเริ่มต้นคือ:
    *   **Username:** `admin`
    *   **Password:** `admin1234`

### ขั้นตอนที่ 4: การเปิดใช้งานเว็บบอร์ด (Web App Deployment)
1.  คลิกปุ่ม **การทำให้ใช้งานได้ (Deploy) > การจัดวางระบบใหม่ (New deployment)**
2.  เลือกประเภทเป็น **เว็บแอป (Web app)**
3.  ตั้งค่าตัวเลือกดังนี้:
    *   **เรียกใช้ในฐานะ (Execute as):** `ฉัน (Me)`
    *   **ผู้มีสิทธิ์เข้าถึง (Who has access):** `ทุกคน (Anyone)`
4.  กดปุ่ม **Deploy**
5.  ให้สิทธิ์การเข้าถึงข้อมูล (Authorize Access) กับบัญชี Google ของคุณเพื่อให้อ่านเขียน Sheets/Drive ได้
6.  คัดลอก URL ของเว็บแอปที่ได้เพื่อนำไปเข้าใช้งานเว็บไซต์

## 🤖 คู่มือการตั้งค่า AI เขียนเนื้อหา & เจนรูปภาพ

เมื่อเข้าสู่ระบบหลังบ้านด้วยบัญชี `admin` แล้ว ให้ไปที่เมนู **"ตั้งค่าโรงเรียน"** ด้านล่างสุด:

1.  **Google Gemini API Key:** ไปสมัครและคัดลอกคีย์ฟรีจาก [Google AI Studio](https://aistudio.google.com/) แล้วนำมาป้อนเพื่อใช้ในการสร้าง/เขียนเนื้อหาข้อความข่าวสารและบทความต่างๆ
2.  **Groq API Key:** ไปสมัครและคัดลอกคีย์จาก [Groq Console](https://console.groq.com/) แล้วนำมาใส่เพื่อเรียกใช้งานโมเดล `llama-3.3-70b-versatile` ที่โดดเด่นในเรื่องความเร็วในการเจนตัวอักษร
3.  **การสลับผู้ให้บริการ (AI Text Writer Provider):** เลือกตัวคัดเลือกระหว่าง **Gemini** หรือ **Groq** ตามที่คุณชอบใช้เขียนบทความข่าวประชาสัมพันธ์
